# 使用快捷指令添加短时记忆

短时记忆提供快捷指令保存的能力，可以通过捷径快速触发保存动作。

演示视频操作过程：

  • 选择需要复制的文本内容
  • 点击复制
  • 双击「辅助触控」按钮
  • 自动触发保存快捷指令操作
