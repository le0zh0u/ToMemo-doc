# 动作库

动作库是ToMemo高效处理文字的核心功能之一。

## 动作使用

### 应用中使用

在短时记忆中，长按需要处理的记录，在出现的菜单栏中选择「小动作」/「动作流」，点击需要的动作。

**执行结果会直接覆盖原来的内容**

### 键盘中使用

进入短时记忆页面，长按需要处理的记录，在出现的菜单栏中选择「小动作」/「动作流」，点击需要的动作。

**执行结果会直接输出，不改变原文**
