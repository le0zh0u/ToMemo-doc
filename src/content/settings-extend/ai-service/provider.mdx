# AI 供应商

ToMemo支持兼容OpenAI接口的AI供应商，同时接入了Google Gemini的API。

## 添加AI供应商

<Image src="/images/ai-service/add-ai-provider.png" alt="添加AI供应商" width={300} height={300} />

### 供应商

为了更快速添加AI供应，提供了不同供应商的基础配置，直接选择一个供应商，会自动填入名称和Base URL。

### Base URL

如果通过选择供应商的方式，会自动填入对应供应商的Base URL，不需要修改。

如果手动添加，直接填写供应商的接口域名地址即可。

### API Key

将AI供应商的后台中，创建API Key，然后完整的内容填入到这里。

> [!TIP]
>
> 如果没有当前供应商的API Key，可以点击输入框左下角的 「API Key」按钮，跳转到供应商后台创建API Key。
>

### 测试

完成Base URL和API Key的填写后，点击「测试」按钮，可以测试是否可以正常连接到AI供应商。

按钮的背后是调用的AI供应商模型列表的接口，接口与OpenAI的模型接口一致。如果部分AI供应商自定义了模型列表的借口，会出现失败的情况，不要担心。

### 添加模型

核心字段是「模型名称」，在这里需要输入当前AI供应商支持的完整的模型名称，比如：`gpt-4o`。

模型名称会在接口调用时发送给AI供应商，如果填入他们无法识别的模型名称会导致调用失败。

> [!TIP]
>
> 为了保证模型名称的准确性，可以点击「加载模型」的按钮，应用会自动拉取模型列表接口，并提供选择
>
> 如果「加载模型」失败，可以通过手动添加，注意模型名称需要要供应商文档中的模型名称保持一致。

### 启用

启用后，可以在「AI助手」中选择当前的AI供应商。
