# 集成DeepSeek

## 添加DeepSeek供应商

进入添加页面后，直接点击「供应商」选项，在里面选择DeepSeek（深度求索），此时，会自动填入名称、Base URL和接口类型。

<Image src="/images/ai-service/deepseek/add-provider.png" alt="添加DeepSeek供应商"  width={300} height={300} />

## 从DeepSeek获取API Key

DeepSeek的API Key获取地址：[点击打开](https://platform.deepseek.com/api_keys)

点击后，完成登录，进入到下面所展示的页面。

点击「创建API Key」按钮，逐步完成创建。

需要**拷贝API Key**， 并妥善保管。

<Image src="/images/ai-service/deepseek/api-key.png" alt="DeepSeek API Key" />


### 拉取模型

将上面获取的API Key，填入到当前页面的API Key输入框中。

点击「拉取模型」按钮，会自动拉取模型。

<Image src="/images/ai-service/deepseek/load-models.png" alt="拉取DeepSeek 模型"  width={300} height={300} />

然后选择需要加载的模型，最后点击「保存」按钮，完成DeepSeek的集成。

## 添加彩虹屁专家🌈

进入「AI助手」页面，开始添加AI助手。

我们从模版中添加「彩虹屁专家」。

<Image src="/images/ai-service/deepseek/ai-assistant.gif" alt="添加AI助手"  width={300} height={300} />




## 键盘中的彩虹屁专家🌈

在键盘中使用「彩虹屁专家」，需要先在「AI助手」中添加「彩虹屁专家」的助手。


<Image src="/images/ai-service/deepseek/keyboard-ai-assistant.gif" alt="键盘中的彩虹屁专家"  width={300} height={300} />
